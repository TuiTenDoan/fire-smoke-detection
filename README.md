# Fire/Smoke Detection System

He thong nhan dien dau hieu khoi/lua tu anh tai len bang Flask + YOLOv8.

## Chuc nang

- Upload anh tinh
- Nhan dien khoi/lua bang YOLO
- Hien thi anh ket qua co bounding box
- Ghi log lich su phan tich
- Khong su dung webcam live

## Cai dat

Chay lenh:

pip install -r requirements.txt

## Chay chuong trinh

Chay lenh:

python app.py

Sau do mo trinh duyet:

http://127.0.0.1:5000
